# 🚀 Markdown to Word Converter

> **Transform your Markdown into professional Word documents with ease!**

A modern, responsive web application that converts Markdown text to professional Word documents with real-time live preview, advanced formatting options, and comprehensive export capabilities.

## ✨ Key Features

### 📝 **Advanced Editor**
- 🎯 Real-time syntax highlighting with live preview
- 🔄 Auto-save functionality - never lose your work
- 🎨 Dark/Light theme toggle
- ⌨️ Comprehensive keyboard shortcuts
- 📊 Live word/character count

### 🎨 **Modern Interface**
- 📱 **Fully responsive design** - Perfect on desktop, tablet, and mobile
- 🖱️ **Intuitive navigation** - Easy-to-use dropdowns and menus
- 💫 **Smooth animations** and professional styling
- 🌈 **Accessible design** with proper contrast and focus states

### 📄 **Multiple Export Formats**
- 📝 **Microsoft Word (.docx)** - Professional document format
- 🌐 **HTML** - Web-ready output with CSS styling
- 🔗 **Standalone HTML** - Self-contained files
- 📑 **PDF** - Print-ready documents (via browser print)
- 📄 **Plain Text** - Simple text format

### 🛠️ **Rich Formatting Tools**
- ✨ **Text formatting** - Bold, italic, strikethrough, code
- 📚 **Headers** (H1-H6) with automatic styling
- 📋 **Lists** - Ordered, unordered, and nested
- 💬 **Blockquotes** and callout boxes
- 🧮 **Tables** with automatic formatting
- 💻 **Code blocks** with syntax highlighting
- 🔗 **Links** with validation

### 📊 **Advanced Features**
- 📈 **Document statistics** - Words, characters, reading time
- 📋 **Template system** - Quick start with professional layouts
- 🔍 **Find and replace** functionality
- 🖨️ **Print preview** capabilities
- 📱 **Offline support** - Works without internet

## 🎯 Quick Start

### 1. **📂 Open the Application**
Visit our [live demo](https://chauhan-mukesh.github.io/Markdown_to_Word/) - works instantly in any modern browser!

### 2. **✍️ Write Your Content**
- Type or paste Markdown in the left editor panel
- Use the formatting toolbar at the bottom for quick formatting
- Watch the live preview update in real-time on the right

### 3. **🎨 Customize Your Document**
- Set document title, author, and date in the navbar
- Choose from pre-built templates using the template dropdown
- Toggle between light and dark themes

### 4. **📥 Export Your Document**
- Click the "Export" dropdown in the navbar
- Choose your preferred format (Word, HTML, PDF, or Text)
- Your document downloads automatically

## 🚀 Local Development

### 📋 **Prerequisites**
- Modern web browser
- Local web server (optional but recommended)

### ⚡ **Quick Setup**
```bash
# Clone the repository
git clone https://github.com/Chauhan-Mukesh/Markdown_to_Word.git

# Navigate to project directory
cd Markdown_to_Word

# Start a local server (choose one):
python -m http.server 8000    # Python 3
python -m SimpleHTTPServer 8000  # Python 2
npx serve .                   # Node.js
php -S localhost:8000         # PHP

# Open in browser
open http://localhost:8000
```
