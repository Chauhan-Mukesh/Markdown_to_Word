## 🎯 Quick Start Tutorial

### 📝 How to Use

1. **📂 Open the Application**
   - Visit our [live demo](https://chauhan-mukesh.github.io/Markdown_to_Word/)
   - No installation required - works instantly in your browser!

2. **✍️ Write Your Content**
   - Type or paste your Markdown in the left editor panel
   - Use the **formatting toolbar** at the bottom for quick formatting
   - Watch the **live preview** update in real-time on the right

3. **🎨 Customize Your Document**
   - Set **document title**, **author**, and **date** in the navbar
   - Choose from **pre-built templates** using the template dropdown
   - Apply **formatting** using toolbar buttons or keyboard shortcuts

4. **👀 Preview & Review**
   - Click **"Preview"** to see your formatted document
   - Use **"Stats"** to analyze word count, reading time, and more
   - **Find & Replace** text using Ctrl+F

5. **📥 Export Your Document**
   - Click **"Export"** dropdown in the navbar
   - Choose your preferred format:
     - 📄 **Word (.docx)** - Professional documents
     - 🌐 **HTML** - Web-ready files
     - 📑 **PDF** - Print-ready documents
     - 📝 **Plain Text** - Simple text files

6. **💾 Save Your Work**
   - Auto-save keeps your work safe
   - Manual save via File → Save
   - Load previous documents via File → Load

## 🚀 Local Development Setup

### 📋 **Prerequisites**
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional but recommended)

### ⚡ **Quick Start**
```bash
# Clone the repository
git clone https://github.com/Chauhan-Mukesh/Markdown_to_Word.git

# Navigate to project directory
cd Markdown_to_Word

# Start a local server (choose one):

# Option 1: Python 3
python -m http.server 8000

# Option 2: Python 2
python -m SimpleHTTPServer 8000

# Option 3: Node.js
npx serve .

# Option 4: PHP
php -S localhost:8000

# Open in browser
open http://localhost:8000
```
