## ‚ùì FAQ & Troubleshooting

### Common Questions

**Q: Do I need to install anything?**  
A: No! MarkdownForge runs entirely in your browser. Just visit the website and start creating.

**Q: Are my documents saved automatically?**  
A: Yes, documents auto-save to your browser's local storage every 30 seconds.

**Q: Can I use it offline?**  
A: Basic functionality works offline after the initial page load.

**Q: Why doesn't my Word export look exactly like the preview?**  
A: Word and browsers render differently. Minor spacing differences are normal.

**Q: Which browsers are supported?**  
A: Chrome 70+, Firefox 65+, Safari 12+, Edge 79+, and mobile browsers.

### Troubleshooting

#### Export Issues
- **Export buttons disabled**: Ensure you have content and click "Preview" first
- **Images missing in exports**: Use absolute URLs, not relative paths
- **Formatting incorrect**: Use standard markdown syntax, avoid complex HTML

#### Performance Issues  
- **Slow preview**: Reduce document size or close other browser tabs
- **App loads slowly**: Check internet connection, clear browser cache

#### Quick Fixes
1. **Refresh the page** - Solves most issues
2. **Clear browser cache** - Fixes loading problems
3. **Try incognito mode** - Rules out browser extensions
4. **Different browser** - Isolates browser-specific issues
